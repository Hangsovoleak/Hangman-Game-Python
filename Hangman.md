# 🎮 Hangman Game (Python)

A simple terminal-based Hangman game built using Python.
This project focuses on basic programming concepts like loops, conditionals, functions, and user input handling.

---

## 🚀 Features

* Random word selection
* User input validation
* Tracks guessed letters
* Limited attempts system
* Win and lose conditions
* Replay option after game ends

---

## 🧠 Concepts Used

* Variables (store game state)
* Lists (word list and guessed letters)
* Conditionals (game logic)
* Loops (game flow)
* Functions (modular structure)

## 🎯 How to Play

1. Run the program
2. Enter your name
3. Guess the word one letter at a time
4. You have 5 chances to guess wrong
5. Win by completing the word before attempts run out

---

## ⚙️ Requirements

* Python 3.x

No external libraries required.
